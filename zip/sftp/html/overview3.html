Overview

    

The Ftps class provides an easy to use programming interface for securely exchanging data with an FTP (File Transfer Protocol) server using SSL (Secure Sockets Layer).  Classes may be found in the com.jscape.inet.ftps package.

 

This section is divided into the following topics.

 

Connecting to an FTP server using SSL

Capturing FTP related events

Setting your directory

Retrieving a directory listing

Uploading files

Downloading files

Remote file operations
