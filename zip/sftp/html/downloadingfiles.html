Downloading files

    

To download a file use one of the available Scp#download methods.

 

Example

 

// set local directory

File localDir = new File("C:/tmp");

scp.setLocalDir(localDir);

 

// download file to local directory

scp.download("/home/jsmith/","file.txt");

 

Example

 

// set local directory

File localDir = new File("C:/tmp");

scp.setLocalDir(localDir);

 

// download file to local directory and save with new name

scp.download("/home/jsmith/","file.txt","filenew.txt");
