Event types

    

The Fxp class is the source for several FXP related events.  All events may be captured by registering your class as a listener by implementing the FxpListener class.  These events are described below.

 

Event

Description

FxpStartEvent

Fired for each file that begins transfer in an FXP transfer.

FxpEndEvent

Fired for each file successfully transferred during an FXP transfer.

FxpFailedEvent

Fired when a FXP transfer has failed to complete.
