Overview

    

The Fxps class is very similar to the Fxp class in functionality with the only difference being that the Fxps class uses secure connections using Ftps instances as the source and destinations servers instead of non-secure Ftp instances.  This is necessary as the commands issued when using a secure connection differ from those when using a non-secure connection.

 

For instructions on using the Fxps class follow those documented in the Fxp class documentation simply replacing each instance of Fxp with Fxps and each instance of Ftp with Ftps in the examples provided.
