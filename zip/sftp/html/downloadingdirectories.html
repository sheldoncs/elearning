Downloading directories

    

To download a directory the Scp#downloadDir method.

 

Example

 

// set local directory

File localDir = new File("C:/tmp");

scp.setLocalDir(localDir);

 

// download remote directory to local directory

scp.downloadDir("/home/jsmith/","images");
