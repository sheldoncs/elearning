Secure FTP Factory

Documentation

The documentation for Secure FTP Factory is split into three sets of documents. These documents are described below.

Name Description
User Guide (CHM) Secure FTP Factory User Guide in compiled HTML format. For Windows platforms only.
User Guide (HTML) Secure FTP Factory User Guide in HTML format.
JavaDoc Generated JavaDoc providing API reference.

Examples

Several console based examples are provided in the examples directory of the Secure FTP Factory distribution. See the README.TXT file for each example for details on it's use.
